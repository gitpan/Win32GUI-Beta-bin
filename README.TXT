This is the binary distribution of the Win32::GUI module.
This file contains only the installable precompiled extension (DLL) 
built with Perl version 5.004_04 and Visual C++ 4.1.

To install the module simply run install.bat.

For docs and samples please download the file Win32Gui-Beta.zip
either from my site or from a CPAN mirror near you.
